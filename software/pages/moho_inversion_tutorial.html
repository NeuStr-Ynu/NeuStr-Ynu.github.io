程序概览

这个程序的核心思想来自 Uieda 与 Barbosa 在 2017 年提出的球坐标系下快速非线性重力反演方法， 但正演部分已经替换为新版球面棱柱体实现。这样做保留了原始方法清晰的反演框架，同时让整体计算速度有了明显提升。

φ(x) = ‖g(x) - g_obs‖² + μ²‖L(x - x₀)‖²

方法来源

Gauss-Newton 迭代、Bott Jacobian 和 Armijo 线搜索的组合框架。

主要改进

使用新版球面棱柱体正演后，模型计算效率比旧实现更高，适合更大区域或更高分辨率试验。

程序结构

MohoLayer 负责建模和正演，MohoInversion 负责反演和结果评估。

适用任务

适合月球或其他行星体壳幔边界反演，也适合做人工模型测试和独立测试点评估。

这套实现并不是重新发明一套反演方法，而是在继承经典文献思路的基础上， 把最耗时的正演部分换成更高效的球面棱柱体正演，因此优势主要体现在运行效率和实际可用性上。

依赖与安装

程序主要依赖以下 Python 库：

numpy
scipy
harmonica
boule
xarray
matplotlib

可以直接使用下面的命令安装：

pip install numpy scipy harmonica boule xarray matplotlib

当前代码默认使用 bl.Moon2015.radius 作为平均半径，所以更偏向月球场景。 如果后续要迁移到地球或其他天体，建议优先检查参考半径、参考面定义和密度差的地球物理意义。

使用流程

  • 准备规则网格的 lon、lat，以及与之对应的二维场数据。
  • 设置观测高度 height、参考莫霍面 reference、初始起伏 surface 和密度差 density_contrast。
  • 创建 MohoLayer 对象，先做一次正演检查量级和符号。
  • 把观测重力场 go 传入 MohoInversion，设置 mu 和 max_iter 后执行反演。
  • 通过残差、收敛曲线和测试点结果评估反演质量。

正则化

mu 越大，模型越平滑；mu = 0 时表示不加正则化。

安全约束

程序内部做了逐点裁剪，确保莫霍面不会越过观测面，这对非均匀参考面特别重要。

最小使用示例

下面这段代码展示了从建模到反演的最小工作流程，你只需要替换成自己的观测数据即可。

import numpy as np

lon = np.linspace(0, 60, 121)
lat = np.linspace(-30, 30, 121)

reference = np.full((lat.size, lon.size), -40000.0)
surface = np.zeros((lat.size, lon.size))
density_contrast = np.full((lat.size, lon.size), 450.0)

height = 10000.0
go = np.zeros((lat.size, lon.size))

layer = MohoLayer(
    lon=lon,
    lat=lat,
    height=height,
    reference=reference,
    surface=surface,
    density_contrast=density_contrast,
)

gz0 = layer.forward(quite=True)

inv = MohoInversion(
    layer=layer,
    go=go,
    max_iter=12,
    mu=0.2,
    quite=True,
)

moho_surface = inv.inversion_gn(gtol=1e-4)
residual = inv.residual_2d()
stats = inv.residual_stats()

print("最终残差统计:", stats)

xarray 输出

程序内置了 xarray 导出接口，方便后续保存和画图：

surface_da = inv.surface_to_xarray()
field_da = inv.field_to_xarray()
residual_da = inv.residual_2d_to_xarray()

测试点评估

如果你有独立测试集，可以直接调用测试点预测接口：

go_pred, residual_test, scores = inv.evaluate_testset(
    lon_pts=test_lon,
    lat_pts=test_lat,
    go_test=test_go,
)

核心类说明

MohoLayer

负责把经纬度网格、参考面和起伏组合成球面棱柱体层，并在规则网格或散点位置计算重力正演值。

MohoInversion

负责一阶差分正则化、Bott 型 Jacobian、高斯-牛顿反演、残差统计和测试集评估。

MohoLayer.forward() 用于规则网格正演，forward_testset() 用于任意散点预测。

MohoInversion.inversion_gn() 是主入口，plot_convergence() 和 plot_residual_histogram() 用于结果诊断。

这版代码里还修复了几个比较关键的稳定性问题，比如 Jacobian 中密度差取绝对值、 非均匀参考面下的逐点安全裁剪，以及最终残差计算流程。这些细节会直接影响反演的收敛行为。

使用建议

  • 先用人工模型做一轮正演-反演闭环测试，确认符号、单位和密度差方向没有问题。
  • 如果结果震荡明显或者高频起伏过强，优先增大 mu，再考虑修改初始模型。
  • 如果网格很大，建议先做低分辨率试算，再逐步提高分辨率。
  • 对不同天体应用时，要同步检查平均半径、参考面定义和密度差含义。
  • 不要只看 RMS 是否下降，最好结合空间残差分布和独立测试点评分一起判断。
