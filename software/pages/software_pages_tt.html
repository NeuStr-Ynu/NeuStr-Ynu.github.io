PlasmaSim

轻量级空间等离子体粒子模拟工具
立即下载 GitHub 仓库

核心功能

粒子-in-cell (PIC) 方法

支持 1D/2D/3D 电磁/静电模拟，高效 Boris 推动器

多种边界条件

周期、吸收、反射、粒子源等多种边界处理方式

实时可视化

内置 Matplotlib / PyVista 支持，密度、场、相空间一键绘制

高性能并行

支持 MPI + OpenMP 混合并行，轻松扩展到数百核

界面预览

主界面：参数设置 + 实时场/粒子可视化

相空间分布 + 能量谱分析

技术栈

Python NumPy / SciPy Numba MPI4Py Matplotlib PyVista OpenMP

获取方式

当前版本：v0.8.2 (2026-03)

GitHub Releases 下载

安装： pip install plasmasim （pip 暂未上架，敬请使用源码安装）
